أداة تحويل XAPK إلى APK و APK إلى EXE

واجهة بسيطة لتسهيل استخراج APK من XAPK وتحويل APK إلى ملف EXE.

فتح المستودع طريقة التشغيل

تحويل XAPK إلى APK

  1. اختر ملف XAPK من جهازك.
  2. حدّد مجلد الحفظ.
  3. سيتم استخراج ملف APK تلقائياً.

تحويل APK إلى EXE

  1. ثبّت PyInstaller عبر pip install pyinstaller.
  2. اختر ملف APK المطلوب.
  3. حدّد مجلد الإخراج لإنشاء ملف EXE.

ملاحظات مهمة

  • تحويل APK إلى EXE مخصص لأجهزة Windows.
  • الملف الناتج يمكنه تثبيت APK عبر adb إن توفر.

طريقة التشغيل

قم باستنساخ المشروع ثم شغّل:

python app.py
