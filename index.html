منصة تحويل XAPK إلى APK و APK إلى EXE

هذه صفحة المشروع الرسمية. يمكنك استخراج APK من ملفات XAPK بسهولة، أو تحويل APK إلى ملف EXE مناسب للتوزيع على Windows.

فتح المستودع طريقة التشغيل

تحويل XAPK إلى APK

  1. شغّل الأداة واختر XAPK من جهازك.
  2. حدّد مجلد الحفظ المناسب.
  3. سيتم استخراج ملف APK مباشرة.

تحويل APK إلى EXE

  1. ثبّت PyInstaller عبر الأمر: pip install pyinstaller.
  2. اختر ملف APK المطلوب.
  3. احصل على ملف EXE جاهز للتوزيع.

ملاحظات مهمة

  • ميزة APK → EXE مخصصة لأجهزة Windows.
  • يمكن تثبيت APK عبر adb إن توفر.
  • شغّل التطبيق عبر: python app.py.
